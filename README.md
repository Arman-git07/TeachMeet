# TeachMeet

This is a modern, feature-rich meeting and classroom application built with Next.js, Firebase, and WebRTC.

## 🚀 How to Publish (Final Steps)

Since you have already set up your Google Cloud Billing and have your API keys, you are **99% finished**. You do **not** need to run any terminal commands like `firebase login` or `firebase deploy`.

Follow these steps in the UI:

1.  **Sidebar (Right side of screen)**: Look for the **"Publish"** tab.
2.  **Step 2 (Billing)**: Click the link/button to select your billing account. (Refresh the page if you don't see it).
3.  **Step 3 (Services)**: Click **"Enable Services"**. This will take ~60 seconds to turn on your database and storage.
4.  **Big Blue Button**: Once Steps 1, 2, and 3 have blue checkmarks, click the **"Publish"** button at the very bottom of that sidebar.

**Your app will be live at:** `https://teachmeet-j7mif.web.app`

## 🛠️ Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + ShadUI
- **Backend**: Firebase (Auth, Firestore, Storage, Realtime Database)
- **P2P Meetings**: WebRTC (Mesh topology)
- **AI**: Google Genkit (Gemini)
