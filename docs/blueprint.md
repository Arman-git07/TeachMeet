# **App Name**: TeachMeet 3D

## Core Features:

- 3D UI: 3D App Interface: Render the TeachMeet UI elements using 3D techniques, with a customizable virtual environment. Includes rendering of user avatars and shared whiteboards.
- Meeting Controls: Meeting Controls: Enable basic controls, including camera on/off, mic on/off, and raise hand, accessible through a streamlined, 3D-rendered interface.
- Screen Sharing & Whiteboard: Screen Sharing & Whiteboard: Integrate a screen-sharing feature along with a collaborative whiteboard with shape recognition and text input. The whiteboard uses the 3D environment as its 'canvas'.
- AI Help Assistant: AI Chat Assistant: A tool providing automated responses to frequently asked questions, assisting users with setup and troubleshooting in a chat-based interface. Save user questions for future improvements.

## Style Guidelines:

- Primary color: A vibrant, saturated green (#32CD32) to evoke a sense of growth and collaboration, contrasting well against the dark background.
- Background color: Dark, desaturated blue (#223D4A) to reduce eye strain and provide a calm, focused environment, consistent with study needs.
- Accent color: A bright, saturated cyan (#00FFFF) to highlight interactive elements, calls to action and important information, drawing user attention without overwhelming the interface.
- Bold, rounded sans-serif fonts to create a modern, friendly interface.
- Use simple, 3D icons for key functions like muting/unmuting, screen sharing, and whiteboard access.
- Keep all buttons and UI elements in the center of the screen for easy reach, ensuring comfortable use across various devices. Left sidebar can slide in with secondary actions. Meeting controls fixed to bottom.
- Use subtle animations like smooth transitions and gentle fades to avoid heavy battery drain. All animations should happen fast and should look organic.
